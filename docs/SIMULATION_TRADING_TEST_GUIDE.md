## 🐛 常见问题排查

### 问题 1：连接上游服务超时

**错误信息**：`timeout of 30000ms exceeded`

**原因**：后端服务未运行或响应过慢

**解决方法**：
1. 确认后端服务已启动：`Get-NetTCPConnection -LocalPort 8000`
2. 查看后端日志是否有错误
3. 重启后端服务

---

### 问题 2：页面白屏或无法访问

**错误信息**：`ERR_CONNECTION_REFUSED`

**原因**：前端服务未运行

**解决方法**：
```powershell
cd apps\dsa-web
npm run dev
```

---

### 问题 3：创建账户失败

**可能原因**：
- 后端数据库未初始化
- 账户 ID 已存在
- 请求参数格式错误

**排查步骤**：
1. 检查后端日志：`logs\stock_analysis_20260516.log`
2. 查看 API 文档确认请求格式
3. 尝试删除旧账户后重新创建

---

### 问题 4：生成建议超时

**可能原因**：
- LLM 服务未配置或响应慢
- 网络问题
- 数据源不可用

**排查步骤**：
1. 检查 `.env` 文件中的 LLM 配置
2. 查看后端日志确认 LLM 调用情况
3. 尝试使用其他数据源

---

### 问题 5：TypeScript 构建错误

**错误信息**：前端构建失败，显示 TypeScript 错误

**说明**：开发模式下不影响使用

**解决方法**：
- 开发模式：忽略这些错误，Vite dev server 仍会运行
- 生产构建：需要修复 TypeScript 类型错误
