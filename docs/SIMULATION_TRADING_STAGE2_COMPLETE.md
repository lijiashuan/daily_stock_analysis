## 🎯 设计亮点

### 1. 多维度选股评分系统
```python
# 评分维度（总分100）
- 流动性（30分）：成交量越大越好
- 波动率（30分）：适中最好（2%-4%）
- 振幅（20分）：日内空间越大越好
- 趋势（10分）：温和上涨最好
- 成交量趋势（10分）：放量最好
```

### 2. 集合竞价智能分析
- 综合买卖盘比例、价格变化、成交量趋势
- 输出置信度，避免盲目跟随
- 生成可操作的交易建议

### 3. 自适应网格策略
- 结合ATR和布林带双重预测
- 根据RSI动态调整订单数量
- 自动计算最优仓位大小

class BaseStrategy(ABC):
