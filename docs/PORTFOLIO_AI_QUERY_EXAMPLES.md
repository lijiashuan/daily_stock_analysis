# AI 持仓查询使用示例

## 正确的提问方式

### ❌ 原来的问题
> "我现在的持仓是什么？"

**问题：** AI可能使用旧的工具，返回昨天的静态数据。

### ✅ 推荐的提问方式

#### 1. 查看当前持仓
```
请帮我查看当前的持仓情况，包括今天录入的交易。
```

AI会自动使用 `get_current_positions` 工具，返回：
- 所有持仓股票及数量
- 今日是否有新交易
- 每个持仓的成本价、最新价、盈亏情况

#### 2. 查看今日交易
```
我今天做了哪些交易？请列出明细。
```

AI会使用 `get_account_recent_trades(days=1)`，返回：
- 今日所有买卖记录
- 每笔交易的价格、数量、金额
- 手续费和税费

#### 3. 复盘今日操作
```
我今天买入了长江电力，卖出了永辉超市，请帮我分析一下今天的操作效果。
```

AI会：
1. 先用 `get_account_recent_trades(days=1)` 获取今日交易
2. 再用 `get_current_positions()` 获取最新持仓
3. 结合实时行情 `get_realtime_quote()` 分析盈亏
4. 给出操作评价和建议

#### 4. 查询某只股票的持仓
```
我现在持有长江电力多少股？成本是多少？
```

AI会：
1. 使用 `get_current_positions()` 获取所有持仓
2. 筛选出长江电力的持仓信息
3. 显示数量、成本价、当前价、盈亏

#### 5. 对比持仓变化
```
我的持仓和上周相比有什么变化？
```

AI会：
1. 使用 `get_current_positions()` 获取当前持仓
2. 使用 `get_account_recent_trades(days=7)` 获取本周交易
3. 分析哪些股票增减仓了
4. 总结变化原因

## 工具对比

| 工具 | 用途 | 适用场景 |
|------|------|----------|
| `get_current_positions` | 获取当前持仓状态 | 查看持有哪些股票、数量、盈亏 |
| `get_account_recent_trades` | 获取近期交易记录 | 复盘操作、核对成交、统计费用 |
| `get_realtime_quote` | 获取实时行情 | 查看当前市场价格 |
| `get_portfolio_snapshot` | 获取完整快照（含风险） | 深度分析、风险评估 |

## 最佳实践

1. **日常查询用 `get_current_positions`**
   - 快速了解持仓状况
   - 包含今日交易
   - 数据全面且清晰

2. **复盘操作用 `get_account_recent_trades`**
   - 查看具体交易明细
   - 统计手续费支出
   - 分析交易频率

3. **估值参考结合使用**
   - 持仓数量来自 `get_current_positions`
   - 实时价格来自 `get_realtime_quote`
   - 手动计算实时市值

4. **定期清理错误数据**
   - 发现错误交易及时删除
   - 重新查询确保持仓正确
   - 保持数据准确性

## 注意事项

⚠️ **价格时效性**
- 持仓中的价格通常是昨天的收盘价
- 如需实时估值，请单独查询实时行情
- 系统会在价格超过2天未更新时给出警告

⚠️ **数据一致性**
- 持仓数量是基于所有历史交易计算的
- 确保交易记录的日期、数量、价格准确
- 错误的交易会导致持仓计算错误

⚠️ **多账户管理**
- 如果有多个账户，注意指定 `account_id`
- 不指定时会汇总所有活跃账户
- 可以分别查询各账户再对比

## 快捷命令

你可以直接使用这些简短的命令：

```
• "持仓" → 查看当前持仓
• "今日交易" → 查看今天做了什么
• "XX股票持仓" → 查看某只股票的持仓
• "本周交易" → 查看本周所有操作
• "持仓盈亏" → 查看盈亏情况
• "刷新持仓" → 重新获取最新持仓
```

AI会根据你的意图自动选择合适的工具。
